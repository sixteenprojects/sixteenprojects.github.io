The Sixteen Project Tool

MITRE ATT&CK TTP Overlap Mapper

Map your target TTPs against 178+ APT groups in the MITRE ATT&CK framework and identify the most likely threat actors.

ATT&CK Enterprise 178+ APT Groups TTP Overlap Scoring Export CSV / Excel / HTML Interactive Dashboard
Download from GitHub

This tool runs as a local web server. After downloading, install the dependencies and run python app.py to launch it on localhost:5000.

How to Start the Tool
1

Open a terminal and navigate to the tool folder:
cd sixteenproject\mitre_ttp_mapper

2

Run the web server:
python app.py

3

The browser will open automatically at localhost:5000, or click Launch Tool above.

4

To stop the server, press Ctrl+C in the terminal.

CLI Usage

Run directly from command line for batch analysis:

python mitre_ttp_mapper.py \
  --ttps-file ttps_cve-2026-41940.txt

python mitre_ttp_mapper.py \
  --ttps T1190 T1059.004 T1505.003 \
  --top 20

python mitre_ttp_mapper.py \
  --ttps-file ttps.txt \
  --force-update
