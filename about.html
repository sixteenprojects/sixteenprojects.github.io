The Sixteen Project

Threat Hunting & Intelligence Platform

What is The Sixteen Project?

The Sixteen Project is an open-source threat intelligence and threat hunting platform built to centralize, enrich, and share knowledge about adversaries, malware, and techniques relevant to the security community. Everything here is built by practitioners — for practitioners.

What's inside?

🌐 TI-Hub — Threat Intelligence Dashboard

A comprehensive, live dashboard aggregating threat intelligence from multiple open-source feeds. Covers 500+ malware families, 100+ threat actors, 60+ ransomware groups, and a global victim feed — all updated daily by an automated pipeline. Features an interactive world map, threat relationship graph, IOC lookups, and statistical analytics.

📄 Threat Reports

In-depth investigation and analysis reports on the latest cyber threats — campaigns, malware families, threat actor operations — researched and written by the team.

🚑 KEV Dashboard

A live intelligence board for CISA's Known Exploited Vulnerabilities catalog. Tracks 1,600+ CVEs with CVSS scores, vendor breakdowns, timeline charts, and threat actor attribution — synced automatically with the CISA KEV feed.

🔎 Threat Hunting Tools

Practical tools and scripts developed in-house to support threat hunting, log analysis, and incident response workflows — including a MITRE ATT&CK technique mapper.

Data Sources

All intelligence data is sourced exclusively from open, publicly available repositories. The platform does not collect or store any private or proprietary data.

Source Used For
Malpedia (Fraunhofer FKIE) Malware families, descriptions, aliases
RansomLook Ransomware groups, live victim feed
OTX AlienVault IOCs (IPs, domains, hashes) per threat actor
CISA KEV Catalog Known exploited vulnerabilities, CVSS, vendors
MITRE ATT&CK Technique mapping, tactic classifications
Natural Earth / D3 Gallery World geodata for the interactive map

Open Source

The Sixteen Project is built entirely with open-source libraries. UI is powered by Bootstrap, Leaflet.js (interactive map), D3.js (threat graph and charts), and vanilla JavaScript — no tracking, no third-party analytics, no dependencies on proprietary platforms. The data pipeline runs on GitHub Actions with daily automated updates.
